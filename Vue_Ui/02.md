# quasar 的一些坑

[quasar 官方](http://quasar-framework.org/)

quasar可以用于快速开发，有Android 与 ios 样式。支持`Electron`与`Cordova`。

使用`quasar-cli`可以快速构建。

坑比`ElementUI`少的多

## 安装

```shell
npm install -g quasar-cli
```

## 样式切换
```
quasar dev mat
quasar dev ios
```

## `.vue`标准格式

```html
<template>

</template>
<script>
export default {
  //
}
</script>
<style>

</style>
```
**注意：使用`quasar`是需要设置引入**

使用定位时
```html
<template>
  <div>
    <div v-ripple>click me</div>
  </div>
</template>
<script>
import { Ripple } from 'quasar'
export default {
  directives: {
    Ripple
  },
  ...
}
</script>
```


使用组件时
```html
<template>
  <div>
    <q-btn @click="doSomething">Do something</q-btn>
    <q-icon name="alarm" />
  </div>
</template>
<script>
import { QBtn, QIcon } from 'quasar'
export default {
  components: {
    QBtn,
    QIcon
  },
  ...
}
</script>
```
